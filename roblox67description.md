# 🌑⚔️ Shadow Realm Roblox Checker ⚔️🌑
**Roblox Account Validator with Capture**
⚡ Fast • Optimized • Stable • Premium

---

## 🚀 Overview
Shadow Realm Roblox Checker is a multi-threaded tool that validates Roblox accounts from combo lists. It checks account validity and captures Robux balance and membership status (Premium/BC) for every valid account, saving categorized results to hit files.

---

## 🎮 Features

### 🎮 Capture Features
- Roblox credential validation
- Robux balance capture
- Premium membership detection
- Builders Club status capture
- Categorized result files

### ⚙️ System Features
- ⚡ Queue-based multi-threaded engine
- 📊 Live CPM and hit counter
- 📁 Categorized hit output files
- 🔄 Auto retry on API errors
- 🌑 Colorized terminal display

